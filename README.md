# LGSI

## 프로젝트에 대한 정보
### 프로젝트 정보
해당 프로젝트를 진행한 단체나 목적에 대해 소개하고, 개발 기간 써준다.
### 팀 소개
팀원들에 대한 소개. 각자 맡은 역할과 소속, 간단한 스펙.
전체적으로 간결하고 명료하게 쓰고, 프로젝트의 가치 전달.

## 시작 가이드
### 요구사항
필요한 요구사항들과 버전들 적어준다.
### 설치 및 실행
repository를 clone하고 패키지 설치, 환경변수 설정, 실행하는 과정에 대한 내용들 코드로.

## 기술 스택
프로젝트 개발하면서 사용한 기술 스택 써준다.
